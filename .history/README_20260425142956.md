# Empirische Studie: Nudging vs. Dark Patterns im Interface-Design

Dieses Repository enthält den Quellcode für die empirische Untersuchung der Bachelorarbeit.
Es handelt sich um eine Webanwendung, die entwickelt wurde, um das Entscheidungsverhalten von Nutzer:innen unter dem Einfluss von Dark Patterns und Nudging-Strategien zu untersuchen.

## Tech Stack

Frontend: React.js (via Vite)
Routing: React Router
Backend & Datenbank: Supabase (PostreSQL, Auth, Row-Level-Security)
Styling: CSS

## Datenbank-Architektur

Das Backend dieser Anwendung wird über den Cloud-Dienst Supabase gehostet. Da aus Datenschutz gründen kein Systemzugriff auf das live Supabase-Dashboard für Dritte gewährt werden kann, ist die vollständige Datenbankarchitektur in diesem Repository dokumentiert.

In der Datei "database.sql" befindet sich ein vollständiges DDL-Skript.Dieses Skript dient als Nachweis für:
-Das relationale Tabellenschema und die Fremschlüsselbeziehungen
-Performance-Indizes und RLS-Policies.
